# Support

## Where To Ask Questions

- Use GitHub Issues for reproducible bugs or concrete feature requests.
- Use GitHub Discussions for product questions, workflow questions, and evaluation ideas.

## What To Include

For package or CLI issues, include:

- the command you ran
- Python version
- install source (`PyPI`, `TestPyPI`, or editable checkout)
- target mode (`external`, `reference`, or `mock`)
- relevant logs or error output

## Security Issues

Please report security-sensitive issues privately through GitHub Security Advisories when available, or contact the maintainer through:

- <https://github.com/NDETERMINA>
