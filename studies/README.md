# Studies

This directory contains the individual studies that make up the larger `Testing ML Systems` project.

Each study is one concrete investigation, with its own code, notebooks, write-ups, and local artifacts.

## Current Studies

- [01-recommender-offline-eval](01-recommender-offline-eval): why offline recommender metrics are useful but incomplete

## Intended Pattern

Each study should keep its own:

- `README.md`
- `src/`
- `notebooks/`
- `docs/`
- local `data/` and `output/` directories when needed

That way the repo can grow part by part without losing the sense that it is still one coherent project.
